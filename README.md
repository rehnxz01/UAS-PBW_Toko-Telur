# Aplikasi Toko Penjualan Telur
Mata Kuliah: Pemrograman Berbasis Web

## Deskripsi
Aplikasi sederhana untuk mengelola data penjualan telur, meliputi daftar barang, harga, stok, pemesanan, dan pengelolaan data oleh admin. Dibangun menggunakan bahasa PHP dan basis data MySQL.

## Teknologi yang Digunakan
- PHP
- MySQL
- HTML & CSS
